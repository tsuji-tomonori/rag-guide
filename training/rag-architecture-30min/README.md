# RAGアーキテクチャ研修（30分）

ソフトウェアエンジニア向けの社内研修資料です。クラウド基盤の一般知識を前提に、`docs/2.アーキテクチャ` を中心として、第1章の導入、テスト、第10章のAWS最小構成を接続しています。

## 成果物

- `deck.md`: 全13枚の画面文言と構成
- `speaker-notes.md`: 約30分の説明台本
- `image-prompts/`: image-genへ渡すページ別プロンプト
- `slides/`: 生成済みスライド画像（16:9 PNG）

## デザイン

- 16:9、オフホワイト `#F7F6F1`
- 主文字 `#2B3A4A`、アクセント `#5E7E96`、補助文字 `#9AA4AD`
- Noto Sans JP相当、フラットな図解、3〜5情報ブロック
- ページ番号は依頼に合わせて右下に `01 / 13` 形式で配置
