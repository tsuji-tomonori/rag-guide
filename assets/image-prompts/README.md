# docs向け図版プロンプトの共通ルール

このディレクトリの番号付きプロンプトは、`docs/` に組み込む16:9図版を再生成するための指示書です。個別ファイルの内容に加えて、次の制約を必ず適用します。

## 画面と余白

- キャンバスは1920×1080とし、背景色 `#F7F6F1` を四辺の端まで途切れずに敷く。
- 指定する120px相当の安全余白は、キャンバス内で文字・図形・矢印を保護する領域であり、背景の外側や内側に別の余白を追加する指定ではない。
- 内側の白いページ・スライド枠、キャンバス外周の枠線、背景を二重に見せる余白を作らない。
- すべての文字、図形、アイコン、矢印、線端を四辺の安全余白内に収め、端で切れたり隠れたりしないようにする。

## 番号と文字

- 右上を含むすべての位置に、ページ番号、スライド番号、図版の通し番号を配置しない。
- 工程の順序、順位、引用IDなど、内容を理解するために意味のある番号は配置してよい。
- `Text (verbatim)` に指定した文字列だけを正確に描画し、ロゴ、透かし、装飾目的の番号を追加しない。

## 生成後の確認

- 背景が画面端まで同色で続き、白い内枠、外枠、二重余白がないことを確認する。
- 四辺と各カードの端を確認し、文字、矢印、図形のクリッピングがないことを確認する。
- 右上だけでなく画面全体を確認し、ページ番号・スライド番号・通し番号がないことを確認する。
