# Example Support Cloud 製品概要

Example Support Cloudは、問い合わせ内容を部門ごとに整理する架空のクラウドサービスです。
この文書に登場する会社、製品、条件は、RAGハンズオンのための例であり、実在しません。

## スタンダードプラン

スタンダードプランでは、50人までの利用者を登録できます。
問い合わせ履歴は90日間保存されます。
管理者はCSV形式で月次レポートを出力できます。

## プレミアムプラン

プレミアムプランでは、500人までの利用者を登録できます。
問い合わせ履歴は365日間保存されます。
監査用レポートと優先サポートを利用できます。

各プランの詳しい利用条件は、対応する規程を参照します。
