# Example Support Cloud サポートポリシー

この文書は、Example Support Cloudのサポート条件の正本です。
内容はRAGハンズオン用の架空情報です。

## スタンダードプラン

サポートの受付時間は、祝日を除く月曜日から金曜日の09:00から17:00までです。
時刻は日本標準時です。
利用者は、管理画面のWebフォームまたは`support@example.invalid`宛てのメールで問い合わせます。
初回応答の目安は2営業日以内です。

## プレミアムプラン

サポートは24時間365日受け付けます。
利用者はWebフォーム、メール、専用電話で問い合わせます。
重大度1の初回応答目標は2時間以内です。

## 問い合わせ時に必要な情報

問い合わせには、契約ID、発生日時、操作手順、表示されたエラーメッセージを含めます。
パスワード、秘密鍵、アクセストークンは送信してはいけません。
