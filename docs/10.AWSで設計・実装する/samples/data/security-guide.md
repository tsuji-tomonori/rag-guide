# Example Support Cloud セキュリティガイド

この文書は、Example Support Cloudの架空のセキュリティ仕様を説明します。

## 暗号化

保存データはAES-256で暗号化されます。
通信にはTLS 1.3を使用します。
秘密鍵はサービスの管理画面へ入力してはいけません。

## バックアップ

構成情報のバックアップは毎日01:00に取得されます。
バックアップの保持期間はスタンダードプランで7日、プレミアムプランで30日です。

## 利用者管理

管理者は、不要になった利用者アカウントを24時間以内に無効化します。
共有アカウントの利用は禁止します。
