# 4. 質問に合う根拠を探す

利用者の質問は、資料に書かれている言葉と同じとは限りません。略語で聞かれることもあれば、会話の流れを前提にした短い質問になることもあります。また、製品名、型番、条文番号のように正確な言葉が重要な場合もあれば、意味が近い説明を幅広く探したい場合もあります。

この章では、質問に対して、どの資料を候補として拾い上げるかを扱います。質問の言葉を整える、言い換えを考える、利用者が見る権限を持つ資料だけに絞る、古い情報を避ける、といった工夫が必要になります。

ここで正しい資料が候補に入らなければ、後でどれだけ回答文を工夫しても、根拠に基づいた答えにはなりません。本章は、回答の土台となる資料探しの考え方を整理する章です。
