# 5. 根拠を選別・整形する

資料を見つけても、そのままAIに渡せば良い回答になるとは限りません。同じ内容の資料が何度も出てきたり、古い資料と新しい資料が混ざったり、質問と少ししか関係のない資料が含まれたりすることがあります。場合によっては、複数の資料が互いに違う内容を示していることもあります。

この章では、見つけた資料をもう一度見直し、回答に使いやすい形へ整える方法を扱います。より関係の深い資料を優先する、重複を減らす、必要な部分だけを分かりやすく並べる、古い情報や信頼しにくい情報を取り除く、といった作業が中心になります。

大切なのは、見つかった資料をただ多く詰め込むことではありません。AIが根拠として使いやすく、利用者にも出どころを確認しやすい形に整えることです。本章では、資料探しで得た候補を回答に使える根拠へ整える考え方を扱います。
