# 8. 安全に運用する

業務でRAGを使うためには、正しい答えを出すだけでは足りません。見せてはいけない資料を見せないこと、個人情報や機密情報を守ること、回答が遅すぎないこと、費用が膨らみすぎないこと、問題が起きたときに原因を追えることが必要です。

この章では、安全性、権限、記録、速さ、安定性、費用、運用ルールをまとめて扱います。利用者の質問、探し出された資料、AIに渡す内容、回答結果、利用記録のすべてが管理の対象になります。

これらは、後から付け足せばよいものではありません。資料を準備する段階、質問に答える段階、運用する段階のすべてに最初から組み込む必要があります。本章は、RAGを安心して使い続けるための土台を整理する章です。
