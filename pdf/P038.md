# 1. はじめに

本論文 **“The Use of MMR, Diversity-Based Reranking for Reordering Documents and Producing Summaries”** は、Jaime Carbonell と Jade Goldstein による SIGIR 1998 の2ページ論文である。検索結果や要約文を「クエリとの関連度」だけで順位付けすると、内容が重複しやすいという問題に対し、関連性と新規性を同時に考慮する **Maximal Marginal Relevance（MMR）** を提案した。([ACM Digital Library][1])

中心的な主張は、検索・要約では「最も関連する情報」を並べるだけでなく、**すでに提示した情報と重複しない、追加的な価値を持つ情報**を優先すべきだというものである。

# 2. 背景

従来の情報検索システムは、各文書とクエリとの類似度を計算し、関連度の高い順に文書を並べる。しかし、同じ出来事を報じるニュース記事などでは、上位文書がほぼ同じ情報を含む可能性がある。その場合、個々の文書は関連していても、検索結果集合全体としては情報量が増えない。

同様の問題は抽出型要約にも存在する。関連度の高い文を単純に選ぶと、導入・結論・複数記事間で同じ内容が繰り返され、限られた要約長を浪費してしまう。

著者らは、ユーザーの情報要求には次の二つがあると考える。

* 特定の話題を深く掘り下げたい場合
* クエリ周辺の異なる話題を広く把握したい場合

MMRは、この「集中」と「多様化」の程度をパラメータで調整できるように設計されている。

# 3. 提案手法

MMRでは、未選択の候補文書 (D_i) について、次の値が最大となるものを逐次選択する。

[
D_i^*=
\arg\max_{D_i\in R\setminus S}
\left[
\lambda,\mathrm{Sim}*1(D_i,Q)
-(1-\lambda)
\max*{D_j\in S}\mathrm{Sim}_2(D_i,D_j)
\right]
]

ここで、

* (Q)：クエリまたはユーザープロファイル
* (R)：通常の検索システムが取得した候補集合
* (S)：すでに選択された文書・文の集合
* (\mathrm{Sim}_1(D_i,Q))：候補とクエリの関連度
* (\mathrm{Sim}_2(D_i,D_j))：候補と既選択項目の類似度
* (\lambda)：関連度と多様性のトレードオフ

である。第1項はクエリ関連度を高め、第2項は既選択項目との重複を罰する。

(\lambda=1) では通常の関連度順ランキングとなり、(\lambda=0) では候補集合内の多様性を最大化する。著者らは、最初は (\lambda=0.3) 程度で話題空間を広く調べ、その後クエリを修正して (\lambda=0.7) 程度で重要部分を深掘りする利用方法を提案している。

要約では、文書を文単位に分割し、各文を候補文書と同様に扱う。類似度にはコサイン類似度を使用し、MMRで選んだ文を最終的には原文中の順序に戻して提示する。

# 4. 実験方法

## 文書ランキング実験

異なる専攻の学部生5名を対象とした小規模なユーザー実験を行った。参加者には通常の関連度ランキングとMMRランキングを匿名の「method R」「method S」として提示し、情報検索タスクを実施させた。最後に、参加者自身が使用したい検索方式を選択した。

## 要約実験

要約については、次の評価が報告されている。

第一に、1998年5月のSUMMAC評価において、15の要約システムと比較した。

第二に、TIPSTERの一つのトピックに属する200記事から50記事を用い、3名の評価者が各文を「関連」「やや関連」「非関連」に分類した。要約長は原文の10%または25%とし、(\lambda=1.0, 0.7, 0.3) および文書冒頭から文を選ぶLead baselineを比較した。

精度は、文書関連性について二つの基準で計算された。

* 基準A：TRECで関連、かつCMU評価者の少なくとも1名が関連と判定した23文書
* 基準B：CMU評価者3名中少なくとも2名が関連と判定した18文書



# 5. 結果

ユーザー実験では、5名中4名、すなわち **80%が最終タスクでMMRを選択**した。参加者は、MMRを話題の探索や候補文書の発見に適していると評価し、関連度のみのランキングは、絞り込んだ話題内で類似文書を探す場合に適していると判断した。5名中3名は、この二つの方式の役割の違いを明確に認識した。

SUMMACでは、MMR要約システムがクエリ指向要約で **F値0.73**、必要な情報が要約に含まれているかを判定する評価で **70%の正解率**を記録し、いずれも参加システム中最高と報告されている。ただし、システム間で要約長などの条件が異なっていたため、著者自身も比較結果は決定的ではないと注意している。

単一文書要約における文精度は次の通りである。数値は「基準A／基準B」を表す。

| 要約長 |          選択方法 |       文精度 |
| --- | ------------: | --------: |
| 10% | (\lambda=1.0) | 0.78／0.83 |
| 10% | (\lambda=0.7) | 0.76／0.83 |
| 10% | (\lambda=0.3) | 0.74／0.79 |
| 10% |          Lead | 0.74／0.83 |
| 25% | (\lambda=1.0) | 0.74／0.76 |
| 25% | (\lambda=0.7) | 0.73／0.74 |
| 25% | (\lambda=0.3) | 0.74／0.76 |
| 25% |          Lead | 0.60／0.65 |

著者らによれば、(\lambda=1.0, 0.7, 0.3) の間に統計的な有意差はなかった。したがって、この実験が示しているのは「多様性を導入しても文精度が大きく低下しなかった」ことであり、MMRが単一文書要約の関連度精度を有意に改善した、という結果ではない。([CMU School of Computer Science][2])

複数文書要約では、ニュース記事群から通常の関連度で取得した上位10パッセージに重複・ほぼ同一の文が多く含まれる一方、MMRはそれらを削減または除去したと報告されている。ただし、この効果に対する定量的な多様性指標は掲載されていない。

# 6. 考察

本論文の最大の貢献は、検索結果集合の品質を、各文書単独の関連度ではなく、**すでに選んだ文書に対して新しい情報をどれだけ追加するか**という集合的な観点から定式化した点である。

特に優れているのは、検索結果の再ランキングと抽出型要約を同一の目的関数で扱えることである。また、既存の検索システムが生成した候補集合に後処理として適用できるため、基盤となる検索モデルを変更する必要がない。

一方、以下の問題が残る。

第一に、文書ランキング実験は参加者5名だけであり、タスク数、クエリ、提示順序の相殺、統計検定などが十分に説明されていない。80%という数値も実質的には4名である。

第二に、単一文書要約ではMMRによる統計的改善は確認されていない。評価されたのは主として関連文の精度であり、MMRの中心的目的である重複削減や情報被覆率が直接測定されていない。

第三に、複数文書要約に関する「明確な優位性」は定性的な報告にとどまり、比較対象、データ数、重複率、統計的有意性が示されていない。

第四に、(\mathrm{Sim}_1) と (\mathrm{Sim}_2) の尺度が異なる場合、そのまま線形結合すると、(\lambda) が意図した意味を持たなくなる可能性がある。類似度の正規化やパラメータ選択方法も必要になる。

さらに、MMRは逐次的な貪欲法であるため、最終的な集合全体の最適性は保証されない。また、重複度を既選択項目との「最大類似度」で表すので、話題全体の被覆や各サブトピックの重要度を明示的にはモデル化していない。

# 7. まとめ

本論文は、検索・要約において関連度のみを最大化するのではなく、既選択情報との重複を減らすべきだと提案した。そのために、クエリ関連度と既選択項目との最大類似度を線形結合したMMRを導入した。

実験規模は小さく、定量的証拠も限定的であるが、次の基本原理を簡潔に示した点が重要である。

> 良い検索結果や要約とは、個々の項目が関連しているだけでなく、各項目がそれまでにない情報を追加する集合である。

# 8. 総評

**査読結果：Weak Accept（弱い採択）**

実際に本論文はSIGIR 1998のプロシーディングスに採択・掲載されている。([ACM Digital Library][1])

独立した査読としても、2ページのアイデア論文としては採択が妥当である。関連度と多様性のトレードオフを、単純かつ汎用的な数式として提示した新規性は非常に高い。一方で、ユーザー実験の規模、比較条件、多文書要約の定量評価は不十分であり、実験結果だけから強い性能優位性を結論づけることはできない。

現代のフルペーパーとして投稿された場合は、より大規模なデータセット、多様性・被覆率を測る指標、強力なベースライン、統計検定、(\lambda) の感度分析を要求し、**Major Revision相当**と判断する。しかし、後続研究につながる明確で再利用可能な概念を提示したという学術的価値は非常に大きい。
